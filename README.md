# Lakers Win Project

A machine learning project to predict Lakers game outcomes.

## Project Structure

```
lakers-win-project/
│
├── backend/          ← Flask app + ML model
│   ├── data/         ← Collected data (CSV files)
│   ├── notebooks/    ← Jupyter notebooks for ML
│   └── app.py        ← Flask API file
│
└── frontend/         ← React app
```

## Backend
- **data/**: Contains CSV files with training data
- **notebooks/**: Jupyter notebooks for data exploration and model training
- **app.py**: Flask API endpoints

## Frontend
- React application for user interface (to be implemented)
